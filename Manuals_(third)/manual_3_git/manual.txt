- Скачиваем git на компьютер и устанавливаем: https://git-scm.com

- Если git установлен, то можно его обновить по команде: git update-git-for-windows

- Настраиваем SSH: https://habr.com/ru/articles/755036/
- Настройка ssh ключей для Windows: https://git-scm.com/book/ru/v2/Git-%D0%BD%D0%B0-%D1%81%D0%B5%D1%80%D0%B2%D0%B5%D1%80%D0%B5-%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F-%D0%BE%D1%82%D0%BA%D1%80%D1%8B%D1%82%D0%BE%D0%B3%D0%BE-SSH-%D0%BA%D0%BB%D1%8E%D1%87%D0%B0

- Может возникнуть ошибка идентификации автора, для этого:
                    git config --global user.email "почта"
                    git config --global user.name "имя"

- Клонируем репозиторий: git clone ssh_строка_с _сайта
- В последующие же разы перед началом работы сначала git pull

- Пишем программу

- Проверяем статус: git status

- Добавляем файлы в git для того, чтобы git начал следить за ними: git add имя_файла

- Проверяем статус: git status

- Для того, чтобы исключить файлы из списка отслеживаемых - создается gitignore: .gitignore

- .gitignore тоже необходимо добавить в git для отслеживания: git add .gitignore

- Фиксируем изменения и сохраняем новую версию кода: git commit -m 'комментарий'

- git push
